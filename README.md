## О проекте

**Console Password Manager** — это утилита командной строки, которая позволяет хранить пароли в *локальном зашифрованном файле*.  
Проект написан на **Python 3.10+** и не требует интернета.

> Миссия проекта: *«Надёжное хранение паролей без облака и лишних сложностей»*.

🔒 Все данные шифруются с использованием библиотеки `cryptography`.

---

## Возможности

- ✅ Добавление, просмотр и удаление паролей  
- 🔐 Шифрование файла с паролями (AES-256)  
- ⚡ Быстрый поиск по сервису  
- 💾 Автоматическое сохранение изменений  
- 🧹 Экспорт в CSV (опционально)

---

## 🧩 Структура проекта

- `main.py` — точка входа
  - `password_manager.py` — основная логика
    - `encryption.py` — модуль шифрования
    - `storage.py` — работа с файлом
- `data/`
  - `passwords.enc` — зашифрованный файл паролей
- `README.md`

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/yourname/console-password-manager.git
   ```

2. Перейдите в папку проекта:
    ```bash
    cd console-password-manager
    ```

3. Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

4. Запустите приложение:
    ```bash
    python main.py
    ```

## Контрольные вопросы

1. **Чем отличается вложенная цитата от обычной?**

Обычная цитата начинается с одного >, вложенная — с двух и более (>>, >>>), что создаёт многоуровневый блок[^1].

---

2. **Как сделать сноску в Markdown и где она отображается?**

В тексте ставится [^ключ], а в конце документа — [^ключ]: текст сноски. Сноски отображаются внизу страницы.

---

3. **В чём разница между - [ ] и - [x] в списках задач?**

- [ ] — невыполненная задача, - [x] — выполненная (чекбокс с галочкой).

---

4. **Какие два способа вставки эмодзи вы знаете?**

    Кодовые последовательности (:smile: → 😄)

    Прямое копирование символов эмодзи (🔐, ✅, ⚡).

---

5. **Зачем нужны HTML-вставки? Приведите пример.**

HTML позволяет добавлять элементы, которых нет в Markdown.
Пример: <kbd>Ctrl</kbd> + <kbd>C</kbd> → отображается как клавиши.

---

6. **Почему важно использовать правильные имена для внутренних ссылок (якорей)?**

Якоря строятся из заголовков (пробелы → дефисы, буквы → нижний регистр). Если имя не совпадёт, ссылка будет вести в никуда.

---

7. **Что произойдёт, если в таблице окажется разное количество столбцов в разных строках?**

Таблица отрендерится с ошибками: лишние ячейки могут быть проигнорированы, а недостающие — останутся пустыми.
